# Specs

- `SPEC-001-dynamic-contract-manager/`: draft v0.2 spec for pre-session contract universe refresh, covering new futures underlyings, new option chains, new strike-price contracts on existing chains, and inactive contract handling. Requirements still contain open clarification items, so implementation is not approved.
- `SPEC-002-dynamic-moneyness-kline-subscriptions/`: approved implementation spec for intraday moneyness-driven Kline filtering, product/contract-level trading-session management, and WebUI trading-time display.
- `SPEC-003-ops-resource-log-monitoring/`: approved/implemented OPS-007 spec for SQLite retry visibility, telemetry retention, disk/CPU/memory monitoring, WAL/log-size alerts, and metrics-worker lock resilience.
